University Placement Suitability Analysis using GIS & Python 

Developed a geospatial analytics and suitability mapping system using Python, GeoPandas, 
Folium, and Matplotlib for university infrastructure planning and accessibility analysis.

• Implemented spatial data analysis, road network accessibility modeling, population density evaluation, 
and service coverage assessment using GIS-based datasets and buffer analysis techniques.

• Built interactive suitability heatmaps, infrastructure visualization dashboards, and recommendation models 
for data-driven urban and educational planning.

• Utilized Python libraries including GeoPandas, Shapely, NumPy, Pandas, and Matplotlib for geospatial processing, 
conformance analysis, and visualization automation.
